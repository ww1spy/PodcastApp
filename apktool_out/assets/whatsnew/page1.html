What is new in BeyondPod 4.2.30

  • Fixed some remaining background service crashes.
  • Fixed lockscreen player not working on some devices running Oreo.
  • Fixed stuck "Loading feeds" notification on Oreo.
  • Fixed notification area/lockscreen player on Oreo.
  • Fixed perceived crash when viewing episode details.
  • Implemented workaround for Chromecast button state. The button is a Google component that is part of the SDK. We use a manual override to default the button color to white when not connected and then change it to green when it is connected. This may change in the future.
